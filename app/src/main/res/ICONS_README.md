# Extension Icons

Place your extension icons in the following directories with these dimensions:

- `mipmap-mdpi/ic_launcher.png` - 48x48 pixels
- `mipmap-hdpi/ic_launcher.png` - 72x72 pixels
- `mipmap-xhdpi/ic_launcher.png` - 96x96 pixels
- `mipmap-xxhdpi/ic_launcher.png` - 144x144 pixels
- `mipmap-xxxhdpi/ic_launcher.png` - 192x192 pixels

## Icon Guidelines

1. Use PNG format with transparency
2. Follow Material Design icon guidelines
3. Make sure the icon is recognizable at small sizes
4. Use colors that represent your source/brand
5. Avoid text in icons (use symbols/logos instead)

## Creating Icons

You can use tools like:
- Android Asset Studio: https://romannurik.github.io/AndroidAssetStudio/
- GIMP or Photoshop for manual creation
- Inkscape for vector-based icons
